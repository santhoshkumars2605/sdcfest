CODEZEELA

Event takes place in Hackerrank. Each question has a pre-determined score. A participant's score depends on the number of test cases successfully passes. Participants are ranked by score. If two or more participants achieve the same score, then the tie is broken by the total time taken to submit the last solution resulting in a higher score.



    RULES

  • Individual participation.
  • Mobile phones are not allowed for any reason.
  • Participants must bring their own laptops.
  • Event coordinators can disqualify and forfeit anyone if they attempt to undermine the contest decorum by getting involved in any malicious activity during any stage of the event.
  • Event consists of 1 round with 3 coding questions categorized as easy, medium, and hard.
  • Event duration would be 1:30 an hour.
